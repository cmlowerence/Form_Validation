Validating Web Forms

First Name must be alphanumeric and should contain 3-16 characters
Last Name must be alphanumeric and should contain 3-16 characters
E-mail must be a valid address. eg: johndoe123@example.com
Password must be alphanumeric (@, _, and - are allowed) and between 8-20 characters
A valid Telephone number (10-11 digit) (without Country Code)
Bio must contain only alphabet, underscore, hyphen and must be of 8-50 characters
